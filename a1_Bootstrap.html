Introduction to Bootstrap

Bootstrap is a framework where you can literally use an automatic set of styles in your code. In this case, you can just link the bootstrap in your HTML file and use a bootstrap class to use these styles.

Containers & breakpoint basics

Example of use: <div class="container"></div>

container = 540px

container-sm = 540px

container-md = 720px

container-lg = 960px

container-xl = 1140px

container-xxl = 1320px
